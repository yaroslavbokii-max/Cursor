## 📋 How to Use Templates

1. **Identify the question type** — Match your question to a template
2. **Customize breakdowns** — Add/remove dimensions based on your data
3. **Select methods** — Use suggested methods or add based on data availability
4. **Define metrics** — Confirm key metrics are available in your data
5. **Generate outputs** — Use template structure for consistent deliverables
