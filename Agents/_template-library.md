## 📊 Presentation Templates

### Executive Update (5 slides)

```
Slide 1: Title + Key Metric
Slide 2: Situation (What's happening)
Slide 3: Key Findings (3 bullets max)
Slide 4: Recommendations
Slide 5: Next Steps + Ask
```

**Best for:** Board updates, leadership meetings
**Time to present:** 10-15 minutes
**Content density:** Executive (40-60 words/slide)

---

### Deep Dive Analysis (12 slides)

```
Slide 1: Title + Agenda
Slide 2: Executive Summary
Slide 3: Context/Background
Slide 4: Methodology
Slide 5-8: Findings (one per slide)
Slide 9: Implications
Slide 10: Recommendations
Slide 11: Risks & Mitigations
Slide 12: Next Steps
```

**Best for:** Quarterly reviews, strategy presentations
**Time to present:** 30-45 minutes
**Content density:** Balanced (80-100 words/slide)

---

### Problem-Solution Deck (8 slides)

```
Slide 1: Hook (Provocative question/stat)
Slide 2: Problem Statement
Slide 3: Impact of Problem
Slide 4: Root Causes
Slide 5: Solution Overview
Slide 6: How It Works
Slide 7: Expected Results
Slide 8: Call to Action
```

**Best for:** Proposals, pitches, change requests
**Time to present:** 15-20 minutes
**Content density:** Executive (40-60 words/slide)

---

### Workshop Presentation (15 slides)

```
Slide 1: Welcome + Objectives
Slide 2: Agenda
Slide 3: Icebreaker Activity
Slides 4-8: Theory Content (one concept per slide)
Slide 9: Transition to Practice
Slides 10-13: Exercise Instructions
Slide 14: Key Takeaways
Slide 15: Resources + Q&A
```

**Best for:** Training, skill development
**Time to present:** 25-30 minutes (+ practice time)
**Content density:** Balanced (80-100 words/slide)

## 🎓 Workshop Templates

### Standard Workshop (60 min)

```
Structure:
├── Opening (5 min)
│   ├── Welcome & Icebreaker
│   └── Objectives & Agenda
├── Theory (20-25 min)
│   ├── Concept 1 (5-7 min)
│   ├── Concept 2 (5-7 min)
│   └── Concept 3 (5-7 min)
├── Practice (25-30 min)
│   ├── Individual Exercise (10 min)
│   ├── Pair/Group Exercise (15 min)
│   └── Debrief (5 min)
└── Close (5 min)
    ├── Key Takeaways
    └── Q&A

Deliverables:
- Presentation slides (10-15)
- Exercise worksheets (2-3 exercises)
- Cheatsheet (1 page)
- Facilitator guide
```

---

### Half-Day Workshop (3 hours)

```
Structure:
├── Opening (15 min)
├── Module 1 (45 min)
│   ├── Theory (20 min)
│   └── Practice (25 min)
├── Break (15 min)
├── Module 2 (45 min)
│   ├── Theory (20 min)
│   └── Practice (25 min)
├── Break (15 min)
├── Module 3 (45 min)
│   ├── Case Study (30 min)
│   └── Debrief (15 min)
└── Close (15 min)
    ├── Action Planning
    └── Q&A

Deliverables:
- Presentation deck (30-40 slides)
- Workbook (10-15 pages)
- Case study materials
- Reference guide
- Facilitator guide
```

---

### Micro-Workshop (30 min)

```
Structure:
├── Hook (2 min)
├── One Key Concept (8 min)
├── Quick Practice (15 min)
└── Takeaway (5 min)

Deliverables:
- Slides (5-7)
- One-page exercise
- Cheatsheet card
```

## 📋 Report Templates

### Weekly Status Report

```
Section 1: Summary (2-3 sentences)
Section 2: Highlights (3-5 bullets)
Section 3: Lowlights/Risks (2-3 bullets)
Section 4: Key Metrics Table
Section 5: Next Week Focus
Section 6: Asks/Blockers
```

**Format:** 1 page
**Update frequency:** Weekly

---

### Monthly Business Review (MBR)

```
Section 1: Executive Summary
Section 2: Performance vs. Goals
Section 3: Key Metrics Dashboard
Section 4: Wins & Achievements
Section 5: Challenges & Actions
Section 6: Focus Areas for Next Month
Section 7: Resource Needs
Appendix: Detailed Data
```

**Format:** 5-10 pages + appendix
**Update frequency:** Monthly

---

### Quarterly Business Review (QBR)

```
Section 1: Executive Summary
Section 2: Quarter in Review
Section 3: Goal Achievement (with visuals)
Section 4: Key Initiatives Status
Section 5: Financial Performance
Section 6: Team & Capacity
Section 7: Customer/Market Insights
Section 8: Risks & Mitigations
Section 9: Next Quarter Priorities
Section 10: Strategic Recommendations
```

**Format:** 15-25 pages
**Update frequency:** Quarterly

## 📝 Document Templates

### One-Pager

```
[TITLE]

THE SITUATION
[2-3 sentences on context]

THE CHALLENGE
[2-3 sentences on problem]

THE SOLUTION
[3-4 bullets on approach]

THE IMPACT
[2-3 bullets on expected results]

THE ASK
[What you need from the reader]
```

---

### Decision Document

```
DECISION REQUIRED: [One-line summary]

CONTEXT
[Background in 2-3 paragraphs]

| Option | Pros | Cons | Cost | Time |
|--------|------|------|------|------|
| A      |      |      |      |      |
| B      |      |      |      |      |
| C      |      |      |      |      |

RECOMMENDATION
[Your recommended option with reasoning]

RISKS
[Key risks and mitigations]

DECISION NEEDED BY: [Date]
```
