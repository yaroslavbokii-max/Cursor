## 📊 DATA & ANALYSIS

### Analyze Data / Find Insights

**Prompt template:**
```
Analyze this [data type] and help me understand [specific question].
Focus on: [what matters most]
Audience: [who will see this]

[Attach your CSV/Excel file]
```

**Example:**
```
Analyze this weekly conversion data and help me understand why conversion 
dropped 15% last week.
Focus on: Root causes and actionable recommendations
Audience: My manager and ops team
```

**What you'll get:**
- Executive summary with key findings
- Visualizations (charts, tables)
- Root cause analysis
- Recommended actions
- Interactive dashboard (optional)

---

### Build a Dashboard

**Prompt template:**
```
Create a dashboard showing [metrics] for [scope].
Key questions to answer: [list 2-3]
Update frequency: [daily/weekly/monthly]

[Attach your data]
```

**Example:**
```
Create a dashboard showing restaurant partner performance for my portfolio.
Key questions: Which partners need attention? Where are we growing?
Update frequency: Weekly
```

**What you'll get:**
- Interactive HTML dashboard
- Key metrics at a glance
- Drill-down capability
- Print-ready version

---

### Weekly/Monthly Report

**Prompt template:**
```
Create my [frequency] [report type] report.
Include: [key sections]
Compared to: [previous period / target]
Highlight: [what matters most]

[Attach your data or point to existing template]
```

**Example:**
```
Create my weekly ops report.
Include: Orders, GMV, conversion, partner health
Compared to: Last week and monthly target
Highlight: Anything that changed more than 10%
```

**What you'll get:**
- Formatted report (your chosen template)
- Narrative commentary
- Key callouts
- Charts and tables

## 📑 PRESENTATIONS & DECKS

### Create a Presentation

**Prompt template:**
```
Create a [length] presentation about [topic].
Audience: [who will see this]
Goal: [what you want them to do/think after]
Tone: [formal/conversational/inspiring]
```

**Example:**
```
Create a 10-slide presentation about our Q4 results.
Audience: Leadership team
Goal: Get approval for next quarter's initiatives
Tone: Professional but confident
```

**What you'll get:**
- Slide deck (HTML + Gamma-ready)
- Speaker notes
- One key message per slide
- Professional visualizations

### Prepare for a Meeting

**Prompt template:**
```
Help me prepare for my [meeting type] with [who].
Context: [what's happening]
My goal: [what I want to achieve]
Concerns: [potential objections or issues]
```

**Example:**
```
Help me prepare for my quarterly review with my restaurant partner.
Context: Their performance dropped 20% this quarter
My goal: Agree on an improvement plan together
Concerns: They might blame our delivery times
```

**What you'll get:**
- Talking points
- Data summary
- Anticipated questions + answers
- Recommended next steps

## ✍️ CREATIVE & CONTENT

### Write a Push Notification Campaign

**Prompt template:**
```
Create a push notification campaign for [goal].
Target audience: [who]
Timing: [when to send]
Tone: [urgent/friendly/exclusive]
Constraints: [character limits, etc.]
```

**Example:**
```
Create a push notification campaign for our new Order Ahead feature.
Target: Users who order lunch 2+ times per week
Timing: 10 AM on weekdays
Tone: Helpful and convenient
Constraints: Max 100 characters
```

**What you'll get:**
- 3-5 notification variants
- A/B testing recommendations
- Suggested send times
- Success metrics to track

## 🔧 PROCESS IMPROVEMENT

### Improve a Process

**Prompt template:**
```
Help me improve our [process name].
Current state: [how it works now]
Problems: [what's not working]
Goal: [what success looks like]
Constraints: [what can't change]
```

**Example:**
```
Help me improve our restaurant onboarding process.
Current state: Takes 14 days average, involves 5 handoffs
Problems: Too slow, partners get frustrated, lots of back-and-forth
Goal: Under 7 days, fewer touchpoints
Constraints: Legal review step must stay
```

**What you'll get:**
- Current state analysis
- Bottleneck identification
- Recommended improvements
- Implementation roadmap

---

### Brainstorm Solutions

**Prompt template:**
```
I'm stuck on [problem]. Help me brainstorm solutions.
Context: [background]
What I've tried: [previous attempts]
Constraints: [limitations]
```

**Example:**
```
I'm stuck on improving courier retention in Zone 5.
Context: 40% churn in last quarter, mostly new couriers
What I've tried: Bonus program, adjusted zones
Constraints: Can't increase base pay
```

**What you'll get:**
- Fresh perspectives
- Structured brainstorm
- Pros/cons analysis
- Recommended next steps
