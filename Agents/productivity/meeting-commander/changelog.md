# Changelog — Meeting Commander

## [1.0.0] — 2026-01-12

### Added
- Initial release
- Multi-format input (transcript, notes, verbal recap)
- SMART action item extraction with owners and deadlines
- Decision logging with rationale and confidence levels
- Follow-up email generation
- 1:1 talking points generation
- Reminder schedule creation
- Meeting type specializations (1:1, Team, Project, Brainstorm, Decision)
- Learning Loop Protocol with retrospective questions
- Integration with project-orchestrator, team-template-generator, people-leader-coach

### Core Features
- Adaptive depth (Quick/Standard/Comprehensive)
- BLUF-first summaries
- Unassigned item flagging
- Missing deadline detection
- Implicit commitment extraction

### Output Formats
- Standard meeting summary
- Follow-up email template
- 1:1 talking points
- Reminder schedules
