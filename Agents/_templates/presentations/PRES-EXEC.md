# 📊 Template: PRES-EXEC

**Duration:** 15 minutes
**Slides:** 8-10
**Best For:** Board meetings, C-suite updates, executive reviews

---

## 📋 Structure

### Slide 1: Title
- Topic
- Date
- Presenter
- Company/Team logo

### Slide 2: Executive Summary
**The 30-second version**
- Top finding
- Key implication
- Recommended action
- One supporting data point

_If they leave after this slide, they got the message._

### Slide 3: Context
**Why this matters now**
- Business context (1-2 sentences)
- Why we're discussing this today
- Scope of analysis

### Slides 4-6: Key Findings
**One insight per slide**

Each slide:
- Headline = Takeaway (not topic)
- Supporting visual (chart, diagram)
- 2-3 bullets max
- Source/footnote

Example headlines:
- ❌ "Q4 Revenue"
- ✅ "Q4 Revenue Exceeded Target by 12%"

### Slide 7: Implications
**So what?**
- Business impact
- Risks if no action
- Opportunities

### Slide 8: Recommendations
**What we should do**
- 2-3 specific actions
- Owners
- Timelines
- Resources needed

### Slide 9: Ask
**What you need from audience**
- Decision needed
- Approval required
- Resources requested

### Slide 10: Appendix Intro (optional)
- "Backup slides available"
- List of additional analyses
- Contact for questions

---

## 🎨 Style Guidelines

### Content Density
- **Max 50 words per slide**
- Headlines: 8-12 words
- Bullets: 3 max per slide
- No paragraphs

### Visual Ratio
- 70% visual / 30% text
- One chart per slide
- Clean, minimal design

### Typography
- Title: 28-32pt
- Headlines: 24-28pt
- Body: 16-18pt
- Footnotes: 10-12pt

### Color Usage
- Primary color for key points
- Gray for supporting info
- Accent color for emphasis (sparingly)

---

## 📊 Chart Guidelines

### Best Charts for Exec
| Data Type | Chart |
|-----------|-------|
| Trend | Line chart (clean, minimal) |
| Comparison | Bar chart (horizontal for readability) |
| Composition | Donut or stacked bar |
| Single metric | Big number + sparkline |

### Chart Rules
- Start Y-axis from zero
- Round numbers (5K, 10K, not 5,247)
- Label directly (no legends if possible)
- Highlight the key takeaway

---

## 🎯 Executive Presentation Principles

### SCR Framework
- **S**ituation: What's the context?
- **C**omplication: What's the problem/opportunity?
- **R**esolution: What should we do?

### Pyramid Principle
- Lead with the answer
- Support with evidence
- Details in appendix

### One Message Rule
- Each slide: ONE main point
- Each section: ONE theme
- Whole deck: ONE recommendation
