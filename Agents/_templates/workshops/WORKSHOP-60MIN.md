# 🎓 Template: WORKSHOP-60MIN

**Duration:** 60 minutes
**Structure:** 25 min theory + 35 min practice
**Best For:** Framework training, skill introduction

---

## 📋 Structure

### Opening (5 min)
| Time | Activity | Notes |
|------|----------|-------|
| 0:00-1:00 | Welcome + Hook | Attention-grabbing stat or question |
| 1:00-3:00 | Objectives | 3 learning outcomes |
| 3:00-5:00 | Agenda | Visual roadmap |

### Theory Block 1 (10 min)
| Time | Activity | Notes |
|------|----------|-------|
| 5:00-8:00 | Core concept | Definition + framework |
| 8:00-12:00 | Examples | 2-3 real-world applications |
| 12:00-15:00 | Anti-patterns | What NOT to do |

### Theory Block 2 (10 min)
| Time | Activity | Notes |
|------|----------|-------|
| 15:00-18:00 | Deep dive | Techniques/methods |
| 18:00-22:00 | Case study | Complete worked example |
| 22:00-25:00 | Quick reference | Summary visual |

### Exercise 1 — Individual (7 min)
| Time | Activity | Notes |
|------|----------|-------|
| 25:00-30:00 | Individual work | Apply to simple scenario |
| 30:00-32:00 | Peer share | Exchange with neighbor |

### Exercise 2 — Group (10 min)
| Time | Activity | Notes |
|------|----------|-------|
| 32:00-39:00 | Group activity | Analyze real examples |
| 39:00-42:00 | Share out | Best insights per group |

### Exercise 3 — Challenge (13 min)
| Time | Activity | Notes |
|------|----------|-------|
| 42:00-52:00 | Transform task | Fix bad example |
| 52:00-55:00 | Best work showcase | Peer voting |

### Wrap-up (5 min)
| Time | Activity | Notes |
|------|----------|-------|
| 55:00-57:00 | Key takeaways | 3-5 main points |
| 57:00-59:00 | Action items | What to do next |
| 59:00-60:00 | Q&A + Close | Final questions |

---

## 📦 Deliverables Checklist

- [ ] **Presentation slides** (15-20 slides)
  - Title slide
  - Objectives
  - Theory slides (8-10)
  - Exercise instructions (3)
  - Summary slide
  - Q&A slide
  - Thank you slide

- [ ] **Exercise worksheets** (6 pages)
  - Exercise 1: Individual practice
  - Exercise 1: Peer review
  - Exercise 2: Group critique
  - Exercise 3: Transform challenge
  - Exercise 3: Full version
  - Self-assessment

- [ ] **Cheat sheet** (A5 or A4)
  - Framework summary
  - Key techniques
  - Examples
  - Common mistakes

- [ ] **Facilitator guide** (optional)
  - Timing notes
  - Speaker notes
  - Backup activities
  - FAQ

---

## 🎨 Slide Guidelines

### Content Density
- **Executive:** 40-60 words/slide
- **Balanced:** 60-80 words/slide (default)
- **Detailed:** 80-100 words/slide

### Visual Balance
- 60% visual, 40% text
- One main point per slide
- Max 4 bullets per slide
- Use icons/emojis for visual interest

### Exercise Slides
- Clear time shown
- Step-by-step instructions
- Expected output defined

---

## 🖨️ Print Requirements

### Slides
- Format: 1 slide per page, landscape
- Page size: A4 or Letter
- Include: Print CSS with proper page breaks

### Worksheets
- Format: A4 portrait
- Include: Writing areas, scoring grids
- Page breaks: Never break tables/cards

### Cheat Sheet
- Format: A5 (or A4 folded)
- Fit on one page (front only or front/back)

---

## 📝 Intake Questions

When using this template, confirm:

1. **Topic:** What framework/skill?
2. **Audience:** Who + their prior knowledge?
3. **Industry context:** For examples (e.g., food delivery)
4. **Brand:** Visual style preference
5. **Content density:** Executive/Balanced/Detailed
6. **Print needed:** Which outputs to print?

---

## 🎯 Success Criteria

A successful 60-minute workshop:
- [ ] Participants can explain the core concept
- [ ] Participants completed all 3 exercises
- [ ] Participants have a cheat sheet for reference
- [ ] Participants know what to do next
- [ ] Net Promoter Score ≥ 8/10
