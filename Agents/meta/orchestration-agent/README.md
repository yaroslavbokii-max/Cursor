# Orchestration Agent — User Guide

## What Is This?

The **Orchestration Agent** is your smart assistant that takes a simple request and automatically creates a complete, professional project for you. Think of it as a project manager that handles all the complex planning and execution behind the scenes.

**In simple terms:** You tell it what you want, and it figures out how to get it done using multiple specialized agents working together.

---

## What Can It Do?

The Orchestration Agent can help you create:

- 📊 **Presentations** — Slide decks on any topic
- 🎓 **Workshops** — Complete training sessions with exercises
- 📝 **Documents** — Reports, guides, and analysis documents
- 🔍 **Research Projects** — Structured research and analysis

**The best part:** You don't need to know which agents to use or how to coordinate them. Just describe what you want, and the Orchestration Agent handles everything.

---

## How It Works

The Orchestration Agent follows a simple 5-step process:

```
1. 📥 You provide a simple request
   ↓
2. 🤔 Agent asks clarifying questions (if needed)
   ↓
3. 📋 Agent creates a complete workflow plan
   ↓
4. ⚙️ Agent automatically executes all steps
   ↓
5. ✅ You receive your finished project + analysis
```

---

## How to Use It

### Step 1: Make Your Request

Simply describe what you want in plain English. Include:
- **What** you want to create (presentation, workshop, etc.)
- **Topic** or subject matter
- **Size/scope** (number of slides, duration, audience size)
- **Audience** (who it's for)

**Example requests:**
- "I want a 10-slide presentation about MECE for 60 minutes for business professionals"
- "Create a workshop on Six Thinking Hats for 20 people, 4 hours long"
- "Make a presentation about project management for my team"

### Step 2: Answer Clarifying Questions

The agent will ask you a few questions to make sure it creates exactly what you need:

- **Objective:** What's the main goal?
- **Audience:** Who is this for? What's their knowledge level?
- **Content:** Any specific examples or topics to include?
- **Output:** What format do you prefer?
- **Quality:** Any special requirements?

**Tip:** The more detail you provide, the better the result. But don't worry — you can always refine later.

### Step 3: Let It Work

Once you confirm, the agent automatically:
- ✅ Plans the entire workflow
- ✅ Calls the right specialized agents in the right order
- ✅ Passes information between agents
- ✅ Validates quality at each step
- ✅ Creates all your files

**You just wait** — the agent handles everything!

### Step 4: Review Your Results

You'll receive:
- ✅ All your requested files (presentations, documents, etc.)
- ✅ A complete workflow plan (saved for reference)
- ✅ A gap analysis (suggestions for improvements)
- ✅ A README file explaining what was created

## Tips for Best Results

### ✅ Do This:
- **Be specific** — Include details like slide count, duration, audience
- **Answer questions** — The clarifying questions help create better results
- **Provide context** — Share any relevant background information
- **Mention preferences** — If you have format or style preferences, mention them

### ❌ Avoid This:
- **Don't be too vague** — "Make a presentation" is less helpful than "Make a 15-slide presentation about X for Y audience"
- **Don't skip questions** — The agent asks for a reason
- **Don't worry about technical details** — The agent handles all the technical coordination

## Common Questions

### Q: Do I need to know which agents to use?
**A:** No! The Orchestration Agent automatically selects and coordinates the right agents for your task.

### Q: Can I customize the workflow?
**A:** Yes! The agent will ask clarifying questions, and you can provide specific requirements or preferences.

### Q: What if I don't like the result?
**A:** You can ask the agent to refine specific parts, or provide feedback for improvements.

### Q: How long does it take?
**A:** It depends on the complexity of your request. Simple presentations might take a few minutes, while comprehensive workshops might take longer. The agent will work through all steps automatically.

### Q: What if something goes wrong?
**A:** The agent will pause and ask for your guidance if it encounters any issues or needs to make critical decisions.

### Q: Can I see what the agent is doing?
**A:** Yes! The agent creates a complete workflow plan document that shows all the steps it's taking.
