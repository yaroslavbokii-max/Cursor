## How to Use This Guide

This document contains all the prompts you can use with the Agent Architect. Each prompt includes:

- **What you ask** — The exact prompt to use
- **What you get** — Expected output description
- **Example output** — A preview of what to expect
- **Complexity** — Beginner / Intermediate / Advanced
- **Time** — Estimated time to complete
- **Tags** — For filtering and discovery

## 🚀 Quick Start — Create Any Agent

### Prompt 1: Basic Agent Creation

| Field | Value |
|-------|-------|
| **Prompt** | "Create an agent for [task]" |
| **What You Get** | Complete agent package with guided intake, framework selection, and all documentation |
| **Example Output** | "Let me help you create a world-class [task] agent. First, I'll ask a few questions to understand your needs..." → Full agent.md + prompt-guide.md + prompt-guide.html |
| **Complexity** | Beginner |
| **Time** | 30-60 min |
| **Tags** | `creation`, `guided`, `any-domain` |

---

### Prompt 2: Rapid Agent Creation

| Field | Value |
|-------|-------|
| **Prompt** | "Create a [role] agent. Focus on [specific task]. Channel [experts]. [Complexity level]. Works with [other agents]." |
| **What You Get** | Agent generated with minimal back-and-forth, using specified parameters |
| **Example Output** | Immediate generation of complete agent package based on provided specifications |
| **Complexity** | Intermediate |
| **Time** | 20-40 min |
| **Tags** | `creation`, `rapid`, `experienced-user` |

**Example:**
```
Create a sales copywriter agent. Focus on cold email outreach. 
Channel Gary Halbert and Ben Settle. Comprehensive complexity. 
Works with research agent and email-sender agent.
```
