## Presentation Patterns

### Pattern 1: Problem → Solution (Standard)
```
Hook → Problem → Context → Insight → Solution → Evidence → CTA
```

### Pattern 2: Situation-Complication-Resolution (McKinsey)
```
Situation → Complication → Resolution → Supporting Points → Recommendation
```

### Pattern 3: What-So What-Now What
```
What (Data/Finding) → So What (Implication) → Now What (Action)
```

### Pattern 4: Before-After-Bridge
```
Before (Current state) → After (Desired state) → Bridge (How to get there)
```
