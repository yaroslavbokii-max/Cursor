# ⛔ Universal Enforcement Template

**Apply this block to ALL agents that don't yet have enforcement.**

Copy the appropriate section below and paste after the YAML header in each agent's `.md` file.

---

## Template A: Analysis Agents

```markdown
## ⛔ MANDATORY INTAKE ENFORCEMENT

> **DO NOT SKIP. Ask these questions BEFORE starting any analysis.**

### Required Questions
1. What question/problem are you solving?
2. What data/inputs do you have?
3. Who is the audience for the output?
4. What output format do you need?
5. Any specific constraints or preferences?

### Pre-Delivery Validation
- [ ] Question answered clearly?
- [ ] Evidence-based conclusions?
- [ ] Actionable insights included?
- [ ] Audience-appropriate?
- [ ] Quality score ≥80%?
```

## Apply Enforcement Script

To apply enforcement to an agent:

1. Open the agent's `.md` file
2. Locate the YAML header (between `---` markers)
3. Add `references: [_shared/_v8-learnings-protocol.md]` to YAML
4. Increment version number
5. Add appropriate template AFTER the YAML block
6. Save file
