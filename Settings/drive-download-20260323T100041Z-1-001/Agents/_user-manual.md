## 🏁 First-Time Setup

On your first use, you'll be asked 5 quick questions:

1. **Visual style** — Choose from 10 presets (Bolt, McKinsey, etc.)
2. **Output formats** — Screen, print, Gamma, or all
3. **Content density** — How much text per slide
4. **Involvement level** — How many questions to ask
5. **Industry context** — To load relevant context files

**Time:** 2 minutes  
**Saved:** Forever (you won't be asked again)

## ⚙️ How It Works

### The Simple Flow

```
Your Request → Orchestrator → Specialists → Quality Check → Output
```

### Behind the Scenes

1. **Orchestrator** understands what you need
2. **Picks specialists** (data, presentation, writing, etc.)
3. **Asks questions** if needed
4. **Creates output** using best practices
5. **Checks quality** (must score 80+ to deliver)
6. **Delivers** in your preferred format

### Quality Standards

Every output is checked for:
- ✓ Content fits properly
- ✓ Professional formatting
- ✓ Print-ready (if needed)
- ✓ Brand consistency
- ✓ Data accuracy

## 💡 Tips for Best Results

### Do This ✅

| Tip | Why |
|-----|-----|
| Be specific about audience | Changes tone and detail level |
| Mention output format | Presentation vs. report vs. dashboard |
| Provide context | "This is for our quarterly review" |
| Share your data | Attach files, paste numbers |
| Give feedback | "Make it shorter", "More charts" |

### Avoid This ❌

| Don't | Instead |
|-------|---------|
| "Make something" | "Make a presentation about X for Y" |
| "Analyze everything" | "Find the top 3 insights" |
| Assume it knows your data | Provide or point to files |

## 📁 File Organization

### Where Things Live

```
/Agents/NEW/           → Agent definitions (don't edit)
/Context/              → Your company context (can add files)
/Projects/             → Your outputs (organized by project)
```

### Adding Company Context

Put files in `/Context/` to make them available:
- Brand guidelines
- Metrics glossaries
- Process documentation
- Templates

The system will automatically use relevant context.

## ❓ FAQ

**Q: Do I need to know which specialist to use?**  
A: No. Just describe your task.

**Q: What if the output isn't right?**  
A: Give feedback. Say what's wrong, and it will improve.

**Q: Can it access our company data?**  
A: Files you provide, plus anything in `/Context/`

**Q: How long do outputs take?**  
A: Simple tasks: 2-5 minutes. Complex: 10-20 minutes.

**Q: What if it can't do something?**  
A: It will tell you, and offer alternatives.
