# Changelog — Report Automator

## [1.0.0] — 2026-01-12

### Added
- Initial release
- Template extraction from example reports
- Multi-format data input (CSV, Google Sheets, copy-paste)
- Narrative generation matching extracted style
- Period comparisons (MoM, QoQ, YoY)
- Learning Loop Protocol with retrospective questions
- Integration with data-analyst, data-visualization-expert, presentation-maker, layout-architect
- McKinsey-quality output standards
- Adaptive clarifying questions (quick vs thorough mode)

### Core Features
- Extract structure and placeholders from example reports
- Map data to template automatically
- Generate narratives that match user's writing style
- Validate data and flag anomalies
- Save templates for reuse

### Integration Points
- Handoff protocols for visualization and layout agents
- Context preservation for multi-agent workflows
