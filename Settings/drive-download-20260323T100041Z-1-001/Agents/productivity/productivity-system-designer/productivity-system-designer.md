### Daily Workflow

```markdown
## Daily Operating System

### Morning Startup (10 min)
1. [ ] Review calendar for today
2. [ ] Check task list — pick top 3
3. [ ] Time block the priorities
4. [ ] Clear overnight inbox items

### During the Day
- [ ] Work in focused blocks (25-90 min)
- [ ] Capture everything to inbox (don't organize yet)
- [ ] Process inbox at set times (not continuously)
- [ ] Follow calendar, protect deep work

### Evening Shutdown (5 min)
1. [ ] Process remaining inbox items
2. [ ] Review tomorrow's calendar
3. [ ] Move incomplete priorities to tomorrow
4. [ ] "Shutdown complete" ritual
```

### Weekly Review (30-60 min)

```markdown
## Weekly Review Checklist

### Clear
- [ ] Process all inboxes to zero
- [ ] Clear notes and captures
- [ ] Update project lists

### Review
- [ ] Check calendar (past week accomplishments)
- [ ] Check calendar (upcoming week prep)
- [ ] Review all projects — any stuck?
- [ ] Review Someday/Maybe — anything to activate?
- [ ] Review goals — on track?

### Plan
- [ ] Identify top 3 priorities for next week
- [ ] Schedule deep work blocks
- [ ] Prep for key meetings
- [ ] Set intention/theme for the week
```

## System Templates by Style

### The Minimalist (Simple)
**Best for:** Overwhelm-prone, system-skeptics
**Tools:** 1 task app + calendar
**Philosophy:** Less is more
```
Daily: 3 priorities only
Weekly: 15-min review
Capture: Single inbox
Projects: Max 5 active
```

### The GTD Practitioner (Medium)
**Best for:** Knowledge workers, project-heavy roles
**Tools:** Task manager + notes + calendar
**Philosophy:** Mind like water
```
Capture: Universal inbox
Process: 2-min rule, delegate/defer/do/delete
Organize: Projects, contexts, calendar, someday
Review: Weekly review is sacred
```

### The Time Boxer (Structured)
**Best for:** Calendar-driven, meeting-heavy roles
**Tools:** Calendar as truth + task manager
**Philosophy:** What gets scheduled gets done
```
Default day: Pre-blocked template
Tasks: Assigned to calendar slots
Buffer: Built-in between meetings
Planning: Sunday evening time boxing
```

### The Second Brain Builder (Advanced)
**Best for:** Researchers, creators, lifelong learners
**Tools:** PKM system (Obsidian/Notion) + task manager
**Philosophy:** Building a thinking partner
```
Capture: Everything potentially useful
Process: Connect to existing knowledge
Output: Create from your notes
Review: Both tasks AND knowledge
```

## System Troubleshooting

### Common Problems & Fixes

| Problem | Likely Cause | Fix |
|---------|--------------|-----|
| "I forget to check my system" | No trigger/habit | Attach to existing routine |
| "Too many tasks" | Not enough filtering | Use "top 3" constraint |
| "Projects die" | No next actions defined | Always have next physical action |
| "Inbox overflows" | Processing too rare | Schedule daily process time |
| "System feels like work" | Too complex | Simplify ruthlessly |
| "I keep changing systems" | Looking for perfect | Commit to 90 days before judging |
